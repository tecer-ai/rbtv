# Brand Prism Framework (Kapferer)

**Purpose:** Define brand identity through Jean-Noël Kapferer's 6-facet prism model, grounded in Brand Archetypes personality and M1/M2 customer evidence.

---

## The 6 Facets

Jean-Noël Kapferer's Brand Identity Prism is a hexagonal model that defines brand identity through six facets arranged along two axes:

- **Vertical Axis:** Sender (brand) vs. Receiver (customer)
- **Horizontal Axis:** Externalization (visible to others) vs. Internalization (felt internally)

### Sender Facets (Brand Side)

| Facet | Definition | Key Question | Input Sources |
|-------|------------|--------------|---------------|
| **Physique** | Tangible, visible characteristics | What does someone notice on first encounter? | Working Backwards, Visual Style from Archetypes |
| **Personality** | Character traits the brand would have as a person | If the brand were a person, how would they act? | Brand Archetypes voice character |
| **Culture** | Values and beliefs the brand embodies | What worldview makes this brand's approach inevitable? | Lean Canvas UVP, Unfair Advantage, founder beliefs |

### Receiver Facets (Customer Side)

| Facet | Definition | Key Question | Input Sources |
|-------|------------|--------------|---------------|
| **Reflection** | How customers appear to others through the brand | What kind of person uses this brand? | JTBD social jobs |
| **Self-Image** | How customers feel internally when using brand | How do I feel about myself using this? | JTBD emotional jobs |

### Bridge Facet

| Facet | Definition | Key Question | Input Sources |
|-------|------------|--------------|---------------|
| **Relationship** | The dynamic between brand and customer | What type of relationship is this? | Archetypes relationship type, JTBD forces |

## Prism Layout

```
           SENDER (BRAND)
    ┌─────────────────────────┐
    │  Physique │ Personality │
    │  (what you │ (how you   │
    │   see)     │   act)     │
    ├─────────────────────────┤
    │       Relationship      │
    │   (brand ↔ customer)    │
    ├─────────────────────────┤
    │ Reflection │ Self-Image │
    │ (how others│ (how I     │
    │  see me)   │   feel)    │
    └─────────────────────────┘
         RECEIVER (CUSTOMER)
```

## Facet Definition Guidelines

### Physique

For digital products, Physique includes:
- Interface aesthetic (clean/dense, minimal/rich)
- Interaction patterns (fast/deliberate, guided/exploratory)
- Visual signature elements (color family, typography style, spatial density)
- Product artifacts (dashboards, reports, notifications, emails)

**Format:** List 4-6 tangible attributes + first-impression sentence.

### Personality

Expand from Brand Archetypes voice character:
- 5-7 personality traits beyond just voice (include behavior)
- For each trait, one behavioral example: "Because we are [trait], we [behavior]"
- 3-5 anti-traits (what the brand explicitly is NOT)

### Culture

Frame as beliefs, not generic values:
- 3-5 core beliefs that drive existence beyond profit
- Connect to archetype motivation, UVP, founder convictions
- Format: "We believe [belief]. That is why we [action]."

**Test:** If a competitor could copy the statement without changing behavior, it's too generic.

### Reflection

Describes how **others** perceive the customer:
- 3-5 statements: "A person who uses [brand] is seen as [perception]"
- Grounded in JTBD social jobs
- Must be about the customer, not the brand

### Self-Image

Describes how the **customer** feels internally:
- 3-5 statements: "When I use [brand], I feel [internal state]"
- Grounded in JTBD emotional jobs
- Connects to Problem-Solution Fit emotional transformation

### Relationship

Describes the brand-customer dynamic:
- Named type (mentor-student, coach-athlete, guide-explorer, partner-partner)
- What brand promises, what brand expects
- Situational behaviors: onboarding, success, failure, departure
- One-paragraph Relationship Statement

## Consistency Testing

Test these facet pairs for contradictions:

| Pair | Contradiction Example |
|------|----------------------|
| Physique ↔ Personality | Cold, corporate visuals with playful personality |
| Personality ↔ Culture | Rebellious character with compliance-focused values |
| Culture ↔ Relationship | Transparency values but relationship withholds information |
| Relationship ↔ Reflection | Protective relationship promising customer independence |
| Reflection ↔ Self-Image | "Cutting-edge" appearance but "confused" feeling |
| Physique ↔ Self-Image | Cluttered interface promising "calm and control" feeling |

---

## Prism Narrative Test

Write one sentence connecting all facets:

> "Our brand looks like [Physique], acts like [Personality], believes [Culture], relates to customers as [Relationship], helps them appear [Reflection], and makes them feel [Self-Image]."

If it reads as a single coherent character, the prism is consistent,

## Prerequisites

Before starting Brand Prism:

- Brand Archetypes (M3 Step 2) complete — provides Personality facet foundation
- M1 Conception complete (JTBD, Working Backwards, Lean Canvas)
- M2 Validation findings available (optional but valuable)

---

## Integration

**Receives from:**
- Brand Archetypes → Personality, Relationship type, Visual style
- JTBD → Emotional jobs (Self-Image), Social jobs (Reflection), Forces
- Working Backwards → Customer narrative, tangible product description
- Lean Canvas → UVP, Unfair Advantage, Customer Segments

**Feeds into:**
- Golden Circle → Culture informs Why
- Brand Positioning → Physique + Personality inform differentiation
- Tone of Voice → Personality traits become tone dimensions
- Messaging Architecture → All facets inform messaging layers
- M4 Design Brief → Physique informs visual identity

---

## Pitfalls

1. **Confusing Personality with Reflection** — Personality is brand character; Reflection is customer appearance
2. **Physique as just logo** — Include full product experience, not just visual identity
3. **Generic Culture values** — "Integrity, innovation" says nothing; specific beliefs drive decisions
4. **Skipping consistency test** — Contradictions become visible to customers even when invisible to founders
5. **Building without Archetypes** — Personality must derive from archetype, not be invented fresh
6. **Ignoring Self-Image** — Deepest loyalty forms from internal feelings, not external perception
