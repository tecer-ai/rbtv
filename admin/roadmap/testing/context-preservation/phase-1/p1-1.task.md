### Phase: Execute

   - **Directory structure:** `.claude/memory/` layout (memory.md, general.md, domain/{topic}.md, tools/{tool}.md)
   - **Session start behavior:** Read `memory.md` at session start; load other files only when relevant
   - **Write immediately:** When agent discovers knowledge worth remembering, write to the correct file immediately — not at session end, not when asked
   - **Entry format:** `date | what | why` — nothing more

   - **Pre-execution check:** Before executing commands, read relevant `tools/*.md` to avoid repeating known failures
   - **Create on demand:** If a file doesn't exist yet, create it
   - **Index discipline:** Keep `memory.md` as a current index with one-line descriptions
   - **Path resolution:** IDE mode at `{project-root}/.claude/memory/`; admin mode at `{rbtv-root}/.claude/memory/`
