# Task p3-1: Create LiteLLM Prompt Caching Patch

## Goal

Create a Python patch script (`_mobile/ops/patches/add-litellm-prompt-caching.py`) that modifies Nanobot's `litellm_provider.py` to add `cache_control_injection_points` for system message caching. This enables Anthropic prompt caching for the ~5,000-7,000 token system prompt, reducing cost by up to 90% on cached content.
