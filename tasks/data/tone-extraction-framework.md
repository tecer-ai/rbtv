# Tone Extraction Framework

Reference data for voice signature extraction. Three analysis dimensions plus validation criteria.

---

## 1. Emotional Tone Analysis

Identify psychological and emotional register:

| Dimension | What to Look For |
|-----------|-----------------|
| Vulnerability level | How openly does writer share struggles, doubts, fears? |
| Self-reflection depth | Introspective, examining internal states? |
| Questioning intensity | Question frequency throughout text? |
| Emotional honesty | Raw vs. filtered, direct vs. diplomatic |
| Perspective stance | First person confession, analytical observation, or mixed? |
| Tone shifts | Consistent tone or varied throughout? |

---

## 2. Structural Pattern Analysis

Document mechanical and organizational patterns:

| Pattern | What to Document |
|---------|-----------------|
| List vs. prose ratio | Predominance of bullets/lists vs. flowing paragraphs |
| Nesting depth | Use of nested sub-bullets and hierarchical organization |
| Sentence length | Short declarative, long complex, or mixed? |
| Question density | Ratio of questions to statements |
| Bold/emphasis usage | Frequency and purpose of **bold**, *italic*, formatting |
| Paragraph structure | Single-line bullets vs. multi-sentence blocks |
| Section organization | Use of headers, separators, thematic grouping |
| Transition style | Explicit connectors vs. implicit flow between ideas |

---

## 3. Vocabulary and Register Analysis

Capture lexical signature:

| Element | What to Capture |
|---------|----------------|
| Formality spectrum | Technical/professional vs. colloquial/casual |
| Domain vocabulary | Repeated terms from specific fields |
| Language mixing | Code-switching between languages or registers |
| Metaphor/imagery | Literal vs. figurative expression |
| Hedge words | Qualifiers (maybe, sometimes, a bit) vs. absolutes |
| First-person intensity | Frequency of "I", "me", "my" |
| Action vs. state verbs | "I did" vs. "I feel" vs. "I am" |

---

## Critical Constraints

When creating voice profile:

| Rule | Description |
|------|-------------|
| No third-person analysis | NEVER write "The writer uses..." — you ARE the voice |
| No feature listing | NEVER catalog ("uses bullets, asks questions") — demonstrate instead |
| Match formatting | If source uses **bold**, do same; if nested bullets, replicate |
| Preserve questions | If source asks rhetorical questions, mirror that pattern |
| Honor emotional authenticity | NEVER sanitize raw honesty, NEVER inflate flatness |
| Maintain language purity | If source mixes languages, do same; if pure, stay pure |
| Avoid meta-commentary | NEVER say "I tend to..." or "My writing style is..." |

---

## What NOT to Do

**Third-person analysis (FORBIDDEN):**
> "The writer frequently uses bullet points and asks many introspective questions. The tone is vulnerable and honest."

**Feature listing (FORBIDDEN):**
> "This voice is characterized by: 1) High question density, 2) Nested bullets, 3) Psychological vocabulary"

**External description (FORBIDDEN):**
> "The author's writing style demonstrates a preference for structured organization combined with emotional openness."

---

## What TO Do

**In-voice demonstration (introspective, question-heavy, list-oriented):**

> Por que eu preciso tanto do resultado? Essa pergunta aparece de novo, volta, insiste - e talvez seja porque:
> - Eu sinto que estou perdendo tempo, ficando para trás
> - Tenho medo de não conseguir suportar o processo no longo prazo
> - Sou impulsivo, decido que quero algo e já quero o resultado final
>
> **O que significa realmente colocar energia nisso?** Significa aceitar que vou errar, que vou ter dias ruins, que **não significa fazer tudo perfeito**.

**In-voice demonstration (analytical, structured, professional):**

> Three primary factors drive the current state: resource constraints, timeline compression, and stakeholder misalignment. Each demands a distinct mitigation approach. The question isn't whether we can deliver — it's which trade-offs we're willing to accept.

---

## Validation Checklist

Before presenting voice profile, verify:

- [ ] Written IN the voice, not ABOUT the voice
- [ ] All three dimensions represented (emotional, structural, vocabulary)
- [ ] Source language matched
- [ ] Formatting preferences mirrored
- [ ] Question patterns preserved (if present in source)
- [ ] Emotional register matches authenticity level
- [ ] No meta-commentary or feature listing
- [ ] Profile is usable as a direct writing guide

---

## Success Criteria

| Criterion | Success Looks Like |
|-----------|-------------------|
| Voice authenticity | Someone reading profile would believe original author wrote it |
| Usability | Profile can be used directly as writing guide for new content |
| Emotional match | Vulnerability, introspection, honesty level matches source |
| Structural match | List/prose ratio, nesting, sentence length patterns replicated |
| Vocabulary match | Formality, domain terms, language mixing preserved |
| No analysis mode | Zero instances of third-person analysis or feature cataloging |
