# Task p6-4: ADD Waitlist Email Capture to robotville.ai Home Page

## Goal

Add a "Join the waitlist" email capture to the robotville.ai home page using Netlify Forms — no backend, honeypot spam protection, success redirect page.

---

| File | Purpose |
|------|---------|

| `_mobile/_docs/netlify-placeholder/index.html` | Home page source — target for waitlist integration |

## Execution Flow

### Phase: Understand

1. Read shape.md for hosting platform decision (Netlify) and deploy method.
2. Identify Netlify Forms integration pattern: `data-netlify="true"`, honeypot field, `action` redirect.

### Phase: Execute

1. **UPDATE** `_mobile/_docs/netlify-placeholder/index.html`:
   - Transform from bare placeholder to branded landing page with value proposition (sourced from founder lean-canvas and working-backwards docs).
   - Add email input + "Join the waitlist" button.
   - Add Netlify Forms attributes: `name="waitlist"`, `data-netlify="true"`, `netlify-honeypot="bot-field"`.
   - Add hidden honeypot field (`bot-field`) to catch bots.
   - Set `action="/success"` for post-submission redirect.
   - Design: dark navy palette (`#0b1120` bg, `#6366f1` accent), responsive, system fonts, no external dependencies, no JavaScript.
2. **CREATE** `_mobile/_docs/netlify-placeholder/success.html` — confirmation page with "You're on the list" message and back link.

### Phase: Validate

1. Page renders correctly and is responsive.
2. Form submits to Netlify (verify in Netlify Forms dashboard after first submission).
3. Honeypot field is invisible to users.
4. Success page displays after submission.

### Phase: Close

1. Append execution entry to shape.md.
2. Present summary to user.
