# Canva Visual Design - Platform Interface Knowledge

**Platform:** Canva (canva.com)
**Feature:** AI-powered visual design platform
**Purpose:** Enable AI agents to guide users through creating visual designs with Canva's AI tools, Brand Kit, and template system.

---

## 1. Interface Overview

### Main Components

| Component | Purpose |
|-----------|---------|
| **Magic Studio** | Hub for all AI-powered design tools |
| **Templates** | Pre-built design layouts (1.6M+ free, 3.6M+ with paid) |
| **Brand Kit** | Centralized brand assets (logos, colors, fonts, guidelines) |
| **Dream Lab** | Advanced AI image generation with iterative refinement |
| **Affinity Suite** | Professional vector, layout, and pixel editing (acquired by Canva) |

## 2. AI Tools (Magic Studio)

### AI Design Tools (included in plan, fair-use limits)

| Tool | What It Does |
|------|-------------|
| **Magic Write** | Generate/rewrite text (captions, letters, paragraphs) with optional brand tone |
| **Magic Edit** | Add, replace, modify photo elements via brush selection and text prompts |
| **Magic Eraser** | Remove unwanted objects; AI fills background |
| **Magic Expand** | Extend images in any direction; fix framing |
| **Magic Grab** | Isolate objects for repositioning/resizing; Grab Text for editable text in photos |
| **Magic Switch** | Convert whiteboards/presentations to documents; translate into 150+ languages |
| **Magic Morph** | Apply textures, hues, patterns, 3D effects to text and images |
| **Magic Animate** | Add animations to designs |
| **Magic Resize** | Resize designs for different formats |
| **Magic Background** | Change or remove backgrounds |

### Premium AI Tools (count toward monthly pooled usage)

| Tool | What It Does |
|------|-------------|
| **Magic Media** | Text-to-image, text-to-graphic, text-to-video (20+ styles: Photo, Vibrant, Concept, Dreamy, etc.) |
| **Dream Lab** | Text-to-image with refinement, iterative edits, "Match Image" for reference-style generation |
| **Magic Layers** | Convert flat PNG/JPG into editable layered designs (March 2026, beta: US/UK/Canada/Australia) |
| **Style Match** | Match visual style across designs |

### AI Image Generation Model

Canva uses its proprietary **Canva Design Model** — a foundational model that produces editable designs with layers and objects (not flat images). Powers Magic Media, Dream Lab, Magic Layers, 3D objects, and Copy Art Style. No confirmed use of DALL-E, Imagen, or Stable Diffusion.

## 3. Affinity Integration

Canva acquired Affinity; now part of the Canva ecosystem.

| Plan | Affinity Access |
|------|----------------|
| **Free** | Core vector, layout, and pixel tools |
| **Pro/Business/Enterprise** | Full Affinity + AI tools (Canva AI Studio) |

### AI Tools in Affinity

Generative Fill/Expand/Edit, Subject Isolation, Background Removal, Image & Vector Generation, Portrait Blur, Portrait Lighting, Colorize, Depth Selection, Super Resolve.

Content stored locally in Affinity; not used for AI training. Export from Affinity to Canva for collaboration and scaling.

## 6. AI Agent Guidance

### When to Recommend Canva

| Use Case | Why Canva |
|----------|----------|
| Pitch decks and presentations | Templates, Brand Kit, Magic Design |
| Social media content at scale | Magic Resize, templates, scheduling |
| Brand-consistent marketing materials | Brand Kit with controls, template locking |
| Quick photo editing without Photoshop | Magic Edit, Eraser, Expand, Grab |
| Team collaboration on visual content | Real-time editing, comments, Brand Kit sharing |

### When NOT to Recommend Canva

| Use Case | Better Alternative |
|----------|-------------------|
| Photorealistic AI image generation | Midjourney, DALL-E via ChatGPT |
| Complex vector illustration | Affinity Designer (standalone) or Illustrator |
| Video editing beyond basic | DaVinci Resolve, Premiere Pro |
| Print-ready production work | Affinity Publisher, InDesign |

### Prompt Engineering for Magic Media / Dream Lab

- Be specific about style: "watercolor illustration" beats "artistic"
- Include composition: "centered portrait, dark background, dramatic lighting"
- Specify aspect ratio and intended use: "Instagram story, vertical, 9:16"
- Use negative constraints: "no text, no borders, no watermark"
- "Match Image" in Dream Lab for style consistency across a series
- Magic Layers can turn flat generated images into editable designs

## 7. Limitations

| Limitation | Impact | Workaround |
|-----------|--------|------------|
| Pooled premium AI usage | Hitting limit blocks all premium AI tools until reset | Prioritize premium AI use; use AI design tools (unlimited) for simpler edits |
| No visible usage meter | Can't track remaining premium AI allowance | Monitor usage manually; expect limits toward end of billing cycle |
| Video clips: 5/month | Very limited video generation | Use external video tools for volume |
| Magic Layers: beta | Single-page PNG/JPG only; US/UK/Canada/Australia initially | Wait for wider rollout |
| Canva Design Model only | No DALL-E or Imagen for different aesthetic | Use ChatGPT (DALL-E) or Midjourney alongside Canva |
