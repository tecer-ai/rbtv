# Pilot task — Manus agentic worker validation

Find the **current USD → BRL exchange rate** (US Dollar to Brazilian Real) from a reliable public source.

Then write a short report (a few lines) that states:
- the current USD→BRL exchange rate (the numeric value),
- the source you used (site name and/or URL),
- the date/time the rate is as-of.

Keep the report concise. This is a simple, real autonomous lookup used to validate the worker end-to-end.
