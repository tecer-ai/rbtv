# Task T1: Stdlib HTTP Server + Open/Save-As JSON API

## Goal

CREATE a Python stdlib HTTP server that serves the app shell, serves the open document's directory under `/doc/`, and exposes a JSON API to open a file (read by path) and save-as (write by path). No Flask.

## Execution Flow

### Phase: Understand
1. MUST read every Context File.
2. Mark T1 in progress in BOTH the plan checkbox and this file's `status`.
3. Confirm: `/app/*` and `/runtime/*` are FIXED static roots (anchored at the app's install dir); `/doc/*` is the MUTABLE root re-pointed to the OPENED file's directory on each `open` so relative assets resolve. The runtime is served from `/runtime/*`, NEVER from `/doc/` (per `01` §8, A10).

### Phase: Execute
1. CREATE `server/server.py`: stdlib `http.server`-based server; routes `GET /app/*` (static shell), `GET /runtime/*` (static edit-runtime from the app's own `runtime/` dir — FIXED root), `GET /doc/*` (files under the current, mutable document directory), `POST /api/open`, `POST /api/save-as`; CLI host/port defaults `127.0.0.1:8765`; `run(host,port)`.
2. CREATE `server/api.py`: `open` handler reads `{path}` from disk → `{html,dir,name}`; `save-as` handler writes `{path,html}` to disk → `{ok,path}`; set the server's `/doc/` base to the opened file's dir; validate readable/writable; never execute file content; reject traversal outside an allowed root if configured.
3. Wire error responses per `01` §7 (404 missing, 500 write fail, JSON `{error}`).

**Discovery Handling:** simple (<5 min) → resolve + note in shape.md; complex → add task, notify.

### Phase: Validate
1. Start the server; `GET /app/` responds.
2. `GET /runtime/js/runtime-main.js` returns the runtime module with a JS content-type, regardless of which (or no) document is open.
3. `POST /api/open` with a fixture path returns its bytes + dir + name; afterwards `/doc/*` resolves under that fixture's dir.
4. `POST /api/save-as` writes a temp file; read it back equal.

### Phase: Close
1. Append any decision/discovery to shape.md (standalone) — or defer to orchestrator if under orchestration.
2. Mark T1 complete in BOTH locations after acceptance passes.
