# Task p3-3: Float-paste + insert-paste logic

## Goal

Create `runtime/js/paste.js` — float-paste under the cursor (no reflow), insert-paste into a flex row/block (reflow, FLIP-animated), grid fallback to float, and whole-slide duplicate. ## Acceptance criteria (owner-observable)
- **C6** float-paste: a copy appears under the cursor; OTHER components do NOT move.
- **C7** insert-paste: into a flex row, siblings rebalance (FLIP); into a grid, falls back to a floating copy.
- **C8** whole-slide: a `wasRegion` clipboard pastes a new top-level region after the current slide.
- **C9 (undo half)** each paste is ONE history command (do = insert + tag + side-effect; undo = remove + revert).

### `pasteFloat(x, y)` (design §7.3, D3/D7)
1. `const slot = get(); if (!slot) return;` (empty clipboard → no-op). `if (slot.wasRegion) return pasteRegion();`
2. `const clone = slot.node.cloneNode(true);` (clone per paste so repeats are distinct).
3. Target slide = `document.elementFromPoint(x, y)?.closest("section,.slide,[data-hyp-region]")`; if none, `regionInView()` (fallback, D3). If still none, return.
4. Capture cascade at push time: `const cascadeAtPush = slot.cascade;`.
5. `clone.style.position = "absolute";`. Capture the slide's positioning side-effect for undo: `const prevPos = slide.style.position;` and inside `do()` set `slide.style.position = "relative"` ONLY when `getComputedStyle(slide).position === "static"`.
6. Push ONE command:
   - `do()`: `cmd.do()` (append clone to the slide — `makePasteCommand(clone, slide.getAttribute("data-hyp-id"), null)`); apply the positioning side-effect; `tag()`; then position: measure `clone.getBoundingClientRect()` + `slide.getBoundingClientRect()`, set `clone.style.left/top` so the clone's centre sits at `(x,y)` minus the slide origin minus half the clone size, plus `cascadeAtPush * 16` px on each axis.
   - `undo()`: `cmd.undo()` (remove clone); restore `slide.style.position = prevPos`.
7. After push: `const id = idOf(clone); if (id) { select(id); mount(id); }`. `bumpCascade();`.

### `pasteIntoLayout(x, y)` (design §7.4, D1/D11)
1. `const slot = get(); if (!slot) return;` `if (slot.wasRegion) return pasteRegion();`
2. `const info = current(); if (!info || !info.hypId) return pasteFloat(x, y);` (nothing selected → float). `const target = byId(info.hypId); if (!target) return pasteFloat(x, y);`
3. **Grid fallback (D11):** `const parent = target.parentElement; const pd = parent ? getComputedStyle(parent).display : "";` if `pd === "grid" || pd === "inline-grid" || roleOf(target) === "grid-child"` → `return pasteFloat(x, y);`
4. `const clone = slot.node.cloneNode(true);` Insert as a real sibling immediately AFTER `target`: `makePasteCommand(clone, parent.getAttribute("data-hyp-id"), target.nextElementSibling?.getAttribute("data-hyp-id") ?? null)`.
5. **FLIP** displaced siblings — mirror `interaction.commitDrop` (anchor below): snapshot the rects of `parent`'s `data-hyp-id` children (EXCLUDING the not-yet-inserted clone) BEFORE; push the command (`do(){ cmd.do(); tag(); }`, `undo(){ cmd.undo(); }`); then in `requestAnimationFrame` invert+play the displaced siblings.
6. `const id = idOf(clone); if (id) select(id);` then `reanchorAfterMove();`.

### `pasteRegion()` (design §7.5, D6) — whole-slide
1. `const slot = get(); const clone = slot.node.cloneNode(true);`
2. Current slide = the selected region (`byId(current()?.hypId)?.closest("section,.slide,[data-hyp-region]")`), else `regionInView()`. Insert as a NEW top-level region AFTER it: `makePasteCommand(clone, null, curRegion?.nextElementSibling?.getAttribute("data-hyp-id") ?? null)` (`null` parent → `document.body`).
3. Push `do(){ cmd.do(); tag(); }` / `undo(){ cmd.undo(); }`; then `const id = idOf(clone); if (id) select(id);`.

### `regionInView()` helper
Return the top-level region (`document.body` child carrying `data-hyp-id`) whose `getBoundingClientRect()` contains the viewport centre (`innerWidth/2, innerHeight/2`); else the first such child; else null.

## Current code — FLIP block to mirror (`runtime/js/interaction.js` `commitDrop`, locate by exact text)
```js
  // FLIP FIRST snapshot — DISPLACED SIBLINGS ONLY. The dragged element is
  // EXCLUDED (R10): it lands instantly at its new flow position; FLIP-animating
  // it would invert it back to its old spot for ~180ms (a visible flash).
  const affected = new Set(
    [...childrenWithHypId(oldParent), ...childrenWithHypId(cls.container)].filter((e) => e !== dragEl)
  );
  const first = new Map();
  for (const elx of affected) first.set(elx, elx.getBoundingClientRect());
```
…and the PLAY block:
```js
  // FLIP PLAY
  requestAnimationFrame(() => {
    for (const [elx, before] of first) {
      const after = elx.getBoundingClientRect();
      const dx = before.left - after.left, dy = before.top - after.top;
      if (dx || dy) {
        elx.style.transition = "none";
        elx.style.transform = `translate(${dx}px, ${dy}px)`;
        requestAnimationFrame(() => {
          elx.style.transition = "transform 180ms ease";
          elx.style.transform = "";
          setTimeout(() => { elx.style.transition = ""; if (elx.style.transform === "") elx.style.removeProperty("transform"); }, 220);
        });
      }
    }
  });
```
For insert-paste the "affected" set is the destination parent's `data-hyp-id` children captured BEFORE the insert (the clone is not yet inserted, so it is naturally excluded). Snapshot FIRST, run the command (insert), then PLAY.
