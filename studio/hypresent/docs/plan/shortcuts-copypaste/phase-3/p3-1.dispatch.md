## Acceptance criteria
- `copy(el)` stores `{ node: <deep clone with EVERY `data-hyp-*` attribute stripped>, wasRegion, cascade: 0 }`; last copy wins.
- `wasRegion === (el.parentElement === document.body)`.
- The stored clone carries NO `data-hyp-id` and NO other `data-hyp-*` marker (so paste re-tags cleanly and carries no editor state, incl. comment anchors).

## Module contract (build to this)
```
import { stripIds } from "./element-registry.js";

export function copy(el)        // clone = el.cloneNode(true); stripIds(clone) THEN remove every remaining data-hyp-* attr from clone + all descendants; slot = { node: clone, wasRegion: el.parentElement === document.body, cascade: 0 }
export function get()           // → the slot object, or null
export function hasContent()    // → boolean
export function bumpCascade()   // slot.cascade++ (no-op if empty) — used by repeated float-paste (D7)
```

Stripping ALL `data-hyp-*` (not just ids): after `stripIds(clone)`, walk `clone` + its descendants (`clone.querySelectorAll('*')`, plus `clone` itself) and for each, remove every attribute whose name starts with `data-hyp-`. (`stripIds` only handles `data-hyp-id`; markers like `data-hyp-fontspan`/`data-hyp-agent`/`data-hyp-hook`/`data-hyp-text` must also go.)
